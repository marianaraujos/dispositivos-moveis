# recicla_the

## Sobre o app: 

O ReciclaTHE é um aplicativo móvel desenvolvido como trabalho final da disciplina Programação para Dispositivos Móveis com o objetivo de facilitar a gestão e visualização dos pontos de reciclagem específicos na cidade de Teresina, Piauí.

Objetivos do ReciclaTHE: O desenvolvimento do ReciclaTHE visa alcançar impactos significativos na gestão de resíduos e no engajamento cívico.
## Sobre o Desenvolvimento e Funcionalidade: 

O aplicativo foi construído utilizando a arquitetura MVC (Model-View-Controller) e a tecnologia Flutter. Sua principal funcionalidade é o mapeamento de pontos de reciclagem.
- Localização Precisa: O ReciclaTHE utiliza a localização do usuário e exibe no mapa de Teresina todos os pontos de reciclagem cadastrados.
- Identificação Rápida: Para facilitar o uso, os pontos são marcados com ícones personalizados.
- Praticidade: O usuário pode se movimentar pela cidade e rapidamente verificar o local mais próximo para o descarte de diversos tipos de resíduos, como papel, plástico, vidro, metal, entre outros.
